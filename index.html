Installation

This Theme is made for Blogger platform only. Theme installation on blogger is very easy task. A beginner with Zero knowledge can easily install theme on Blogger.

Step By Step

Here is a brief step by step on how to install Blogger Theme:

  1. Unzip the “Moveone.zip”.
  2. It will contain “Template.xml” File.
  3. Drop it to Desktop.
  4. On Blogger Dashboard Click Template.
  5. Then click Backup/Restore (Top Right).
  6. Now Choose File
  7. Upload “Template.xml” file from Desktop.
  8. Then Click Upload.
  9. Done.

Dafault Settings

After installing the theme, this it the minimun setting to make all widget works fine in this themes.

Embedded Comment

This to make comment area like the Demo and make Emoticon show

  1. On Blogger Dashboard Click Settings
  2. Click Posts and Comments.
  3. in Comment Location Choose Embedded
  4. Save settings

Blog Feed

This to make recent comment, Search Result, Featured Post, and Post by Label work fine

  1. On Blogger Dashboard Click Settings
  2. Click Other.
  3. in Site Feed Allow Blog Feed Choose Full Or Short
  4. Save settings

NOTE:
1. If yours post have image uploaded on blogspot you can choose SHORT. but if yours post just have image from third party image hosting you must choose Full.
2. Blogger feed must be not redirect to other feed site like Feedburner
3. Your's blog or website not Private

Navigation Setup

Main Menu


To Edit Menu in this theme you can follow this step:

  1. On Blogger Dashboard Click Layout
  2. Click PAGES
  3. Add/remov pages according to yourself.

Date Settings



To Edit Date Date (Month Name) in this theme you can follow this step:

  1. On Blogger Dashboard Click Settings
  2. Click Language and formatting
  3. Set Timestamp setting to DAY, MONTH DATE, YEAR (As in above picture)
  4. Save settings

Hide Projects



To Hide Projects in this theme you can follow this step:

  1. On Blogger Dashboard Click Layout
  2. Click PROJECTS
  3. Click EDIT and type: none
  4. Save

Project Label



To Edit Project Label in this theme you can follow this step:

  1. On Blogger Dashboard Click Layout
  2. Click Project Label
  3. Click EDIT and type you Label in Content
  4. Save

Traslator



To Traslate the words in this theme you can follow this step:

  1. On Blogger Dashboard Click Layout
  2. Click Traslator
  3. Now Copy any English Word from the Temlplate i.e. My Diary
  4. Type it in New Site Name Value and your own word in New Site URL value
  5. Now click on Add Link
  6. Save

Follow Widget



To Traslate the words in this theme you can follow this step:

  1. On Blogger Dashboard Click Layout
  2. Click Traslator
  3. Copy any Font Awesome Icon Code, like this: fa fa-facebook
  4. Type it in New Site Name Value and add your link in New Site URL value
  5. Now click on Add Link
  6. Save
